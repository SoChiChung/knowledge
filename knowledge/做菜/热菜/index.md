# 热菜大全

## 窝蛋香菇肉碎西洋菜

**制作步骤**
1. 预处理食材：西洋菜去掉老梗、黄叶，清水浸泡冲洗去除涩味；干香菇温水泡发，挤干水分切成小丁；猪肉剁成肉碎；生姜切丝、胡萝卜切小丁备用。
2. 煸炒底料：锅中倒入少许食用油，下姜丝、香菇丁小火翻炒，炒出香菇鲜香，再倒入肉碎，中火翻炒至肉碎完全散开、变色。
3. 熬煮底汤：放入胡萝卜丁翻炒片刻，加入足量清水，开大火烧至沸腾。
4. 烫煮西洋菜：水开后放入西洋菜，滚煮1-2分钟至菜叶变软。
5. 基础调味：放入适量食盐、鸡粉或松鲜鲜搅拌均匀。
6. 窝蛋焖制：调至最小火，在汤面空位打入完整鸡蛋，不要搅动，盖上锅盖焖至蛋清凝固，想要溏心蛋黄可缩短焖煮时间。
7. 增香出锅：淋入蒜头油拌匀，既可直接喝汤，也能搭配面条食用。

**食材原料**
- 西洋菜：300g
- 猪肉碎：150g
- 干香菇：3～4朵
- 鸡蛋：1个
- 生姜：少许
- 胡萝卜：少量
- 蒜头油、食盐、鸡粉（松鲜鲜）、食用油：适量

## 芥末虾

**制作步骤**
1. 鲜虾剪去虾须、虾头尖刺，开背去除虾线，洗净后沥干表面水分。
2. 腌制虾肉：虾中加入盐、鸡蛋黄、胡椒粉、料酒、干淀粉抓拌均匀，腌制10分钟入味。
3. 锅中倒油烧至六成油温，下入腌好的虾，炸至外壳金黄酥脆后捞出控油。
4. 调芥末酱汁：碗中放入沙拉酱、黄芥末、炼乳、蜂蜜芥末酱，充分搅拌混合顺滑。
5. 将炸好的虾倒入酱汁内翻拌，让每只虾均匀裹满酱汁即可装盘。

**食材原料**
- 大虾：300g
- 鸡蛋黄：1个
- 沙拉酱：4勺
- 黄芥末：适量
- 炼乳：1勺
- 蜂蜜芥末酱：半勺
- 玉米干淀粉：适量
- 料酒、食盐、白胡椒粉、食用油：适量
